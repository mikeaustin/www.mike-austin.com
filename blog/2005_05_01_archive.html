Playing with FreeBSD and KDE


I've been playing around with PC-BSD, which is a more-gui friendly version of FreeBSD. I know Linux already so it didn't take much to learn. I'm starting to like it - and KDE is a sweet desktop. The picture to the left is KMail, KDE's mail application. What I like about BSD is the 'ports'. I can install Firefox simply by typing "add_pkg -r firefox" - it goes off and downloads the dependencies and executables and installs it. Too easy. I may switch over some day from Windows, the only thing stopping me is apps I need such as Photoshop, Dreamweaver. OpenOffice suffices for what I do with MS Office.

New VMWare is pretty nice

I have put myself on a self teaching course to get my skills up to date, and VMWare is a great tool to experiment with different OSs. I've installed and use Windows 2003 Server evaluation, and plan to install FreeBSD, Linux, and maybe Solaris 10. VMWare is *fast*!. It's not simply a virtual machine, it's a hardware virtualizer. Programs are executed natively inside the vm, and only when a program executes supervisor instructions or accesses the hardware does the VM need to intervene. I love not having to dual boot and partition drives and all that mess.
