Impulse

A small, dynamic, prototypical language with planned support for multi-methods and predicate dispatching.

Impulse is a small, dynamic, prototypical programming language with a few unique features. It borrows syntax and semantics from other languages such as Ruby, Smalltalk, Dylan, Io, Lua and JavaScript along with ideas from functional languages.
A core goal is too keep the language, both the implementation and the syntax, small and lightweight. Impulse tries to factor special syntax such as pattern matching and list comprehension into the language using predicates.

Features

Prototype Based
Singletons are easy to create, and objects can change behavior by switching their parent prototypes.
Small and lightweight
The core is less than 2,000 lines of lightweight C++, including the VM, parser and garbage collector.
Reflective and Open
All objects, including native C++ prototypes can be extended and modified. Everything is a first class object.
Easily Embeddable
The language was designed from the bottom up, to provide the most simple interface to the runtime.

Examples

# Get a list of even numbers between 1 and 10
(1..10).map |n| n even ? n * n

? without the : is a binary conditional operator which return void when false. Void is similar to JavaScript's "undefined" and represents "no value", vs nil which means "empty value". 

# Ubiquitous factorial example    
factorial = |n| n == 1 ? 1 : factorial [n - 1]

# A newline after block params defines a multi-line block
factorial = |n|
  n == 1 ? 1 : factorial [n - 1]
end

Download

Source  –  https://github.com/mikeaustin/impulse-language

Compiling should be as easy as "make && sudo make install". Impulse has been tested with GCC 4.3.3 on Ubuntu 32/64, GCC 4.0.1 on MacOS X 10.5, MinGW GCC 3.4.5 on Windows and GCC 3.4.3 on SunOS 5.
