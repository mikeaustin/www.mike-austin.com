Review Invitations - Workgroups
This page allows you to review, expire and re-send the invitations that you have sent.


Click Details on an item to view it’s details.

  • Click Re-send to re-send the invitation in case the member did not respond or receive the invitation.
  • Click Expire Now to expire the invitation.  This disallows any attempts to join the workgroup using this invitation.

Click Done to return to the previous page.
