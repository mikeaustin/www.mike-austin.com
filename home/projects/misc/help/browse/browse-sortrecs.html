Sort Records - DatabasePlus
This page allows you to change the sort order of fields.


To sort in ascending or descending order, add a field to the Sort Order list and select it. Click on Reverse Sort Order to change the sort direction (ascending/descending). A caret(^) to the right of the selected field denotes ascending order. No caret denotes descending order.

To add a field:

  • Select the field from the Available Fields list and click Add Field.

To remove a field:

  • Select the field from the Sort Order list and click Remove Field.

To arrange the sort order of fields:

  • To move the field up one item, select the field from the Sort Order list and click Move Up.
  • To move the field down one item, select the field from the Sort Order list and click Move Down.
  • To reverse the order of the fields, click Reverse Sort Order.

Click Save to save your changes and return to the previous page. Or, click Cancel to abandon them and return to the previous page.
