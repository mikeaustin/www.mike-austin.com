Author Database - DatabasePlus
This page provides buttons to modify settings, security, value lists and to publish reports.


From here, you can go to:

  • Basic Settings
    Allows you to edit the name and description of the database.
  • Advanced Settings
    Allows you to select the date, time, currency and number formats of this database and turn copy protection on or off.
  • Access Control
    Control access to the database by granting permissions to users and groups.
  • Value Lists
    Create or maintain your own set of value list pop-ups for the database.
  • Publish Reports
    Publish reports by creating a URL to embed in a web page or by sending an email to your colleagues.
