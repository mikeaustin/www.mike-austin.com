Author Table - DatabasePlus
This screen enables you to change basic properties of your table, such as the name and description.


From here, you can go to:

  • Basic Settings
    Manage the name and description of this table.
  • Table Fields
    Manage the data elements, or fields, in this table.
  • Access Control
    Control access to this table by granting permissions to users and groups.
  • Table Events
    Define e-mail alerts to trigger when certain actions such as adds, deletes, or modifications occur to the table.
  • Delete Table
    Permanently remove this table and all it’s records from the database. WARNING: This will permanently delete all data in the table.
