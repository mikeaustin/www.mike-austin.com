Edit Calculation - Views
The calculation editor allows you to create dynamic fields build upon functions, fields and operators.


The Calculation Formula text box is the formula editor. You may enter expressions manually, or use the formula tools to create your formulas.

Select the result type from the Result Field Type pull-down menu (Text, Number, True/False, Date, Time, Currency, HTML, URL). This will be used as the field type.

To use the formula tools, follow these hints:

  • To insert a Function, select it from the Functions pull-down menu and click Enter Formula.

  • To insert a field, select if from the Fields pull-down menu and click Insert Field.

  • To insert an Operator, simply click on the operator's button.

Click Done to save your changes, or Cancel to abandon them. You will then be brought back to the Fields Editor.
