Author Report - DatabasePlus
This page allows you to create, modify and delete reports.

From here, you can select:
  • Basic Settings
    Manager the name and description of this report.
  • Fields in Tabs
    Specify which fields are in which tabs, and which report to use for one-to-many fields.
  • Fields
    Specify the field label, presentation, and alignment properties. Also specify which report to use for one-to-one fields.
  • Filter
    Filter records by specifying search criteria.
  • Sort
    Specify the sort order of the fields for this report.
  • Tabs
    Add, modify, delete, or reorder tabs on this report.
  • Delete Report
    Delete this report and permanently remove it from this database.
