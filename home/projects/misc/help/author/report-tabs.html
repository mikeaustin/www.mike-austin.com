Report Tabs - Reports
This page allows you to add, modify, delete, or reorder tabs on the report.


The summary tab is used as the default for tables, views and reports that display a one-to-many relationship with other tables, views, and reports.

Select a tab by clicking on the name in the Current Tabs in Report list. Click Add to create a new tab.

Tab properties:

  • Tab Name - The name of the tab.

Tab positions:

  • Move Up - Moves the position of the tab up one position. On the report, the tab will move to the left.
  • Move Down - Moves the position of the tab down one position. On the report, the tab will move o the right.

Click Save to save your changes, or Cancel to discard them and return to the previous page.
