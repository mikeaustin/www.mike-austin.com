DatabasePlus Overview
 


Navigation Bar

The navigation bar allows you to quickly jump from authoring and browsing your tables, views, and reports.

Workgroups

A workgroup is a list of Bitlocker members. Using workgroups allows you to control who can see your DatabasePlus databases.

Access Control

Access Control allows you to choose who sees what and who can modify what.

Value Lists

Value Lists allow you to create pull-down menu choices. For example, a list of states.

Tables and Fields

In a database, data is stored in two dimensional grids, called tables. Each table contains records that represent an item such as an employee or time sheet. A record is defined by it's fields, which store bits of information such as a name or phone number. Tables, records and fields are similar to that of a spreadsheet's sheets, rows and columns.

Data in one table is often related to data in another. This is called a relationship.

calculated fields. table events. field types. linking tables.

Views and Reports

Views and reports allow you to browse and modify your data. They allow you to search, modify and delete records.
