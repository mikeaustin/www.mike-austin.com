Not Just ‘Another UI’

  • Bring the simplicity and integration of PDAs to the desktop
  • Standardized information sharing
    • Contacts, Calendar, Tasks, Projects
  • Lightweight database, binary xml
    • Persistent storage available to any application
  • Virtual desktop Workspaces
    • Configurable categories, filtered database items
  • Highly reusable interface components
    • Buttons, Tree, List, Timeline, Property

Inertia Desktop

Tablets, Workspaces and Projects

  • Tablets
    • Tabbed windows arranged in a virtual shelving system. Popup Tablets allow quick access to global tools.
  • Workspaces
    • Virtual desktops that contain Tablets and are associated with multiple projects and categories.
  • Projects
    • Projects are a collection of related items such as messages, contacts, documents, notes, etc.

Tablets

  • Used for most ‘container’ visuals
    • Menus
    • Windows
    • Tabbed Dialogs
  • Tablets can be dragged by detach bar to copy/detach

System Popup Tablets

  • Examples of System Tablets:
    • Calculator
    • Spell Checker
    • Character Map
    • Thesaurus
  • Also, can give quick access to
    • Contacts
    • Messages
    • Tasks
    • Documents
    • Notes

Menus and Menu Items

  • Submenu’s can be expanded inline, and detached to create a Tablet

Dialog Windows
