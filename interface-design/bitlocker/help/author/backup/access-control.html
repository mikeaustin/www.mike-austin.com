Modify Access Rights

Status

·         Enabled
Gives the user access to the database or table.

·         Disabled
Prohibits the user from any access to the database or table.


Access Rights

  • Can ADD new records
    Allows the user to add records to the database or table.
    NONE – cannot view, modify or delete records

  • Can VIEW records
    Allows the user to view records in the database or table.
  • Can MODIFY records
    Allows the user to modify records in the database or table.
  • Can MODIFY or DELETE records
    Allows the user to modify or delete records in the database or table.
