Go to start of metadata

Our communications systems and development tools affect the way we work.  Following is a list of everything we use to get the job done.

Communication Systems

Google Gmail

Gmail is used for broadcasting information and keeping everyone informed.

  • Go to Settings > Labels to organize your new and existing email
  • Go to Settings > Labs and try out the Preview Pane and more

Google Groups

Mailing lists are actually Google Groups, so you can join and view public ones.

  • You can visit all Storm8 public groups http://ml.storm8.com
  • If your Group is public, make sure to include it in the directory!
    Manage > Information > Directory | List this group in the directory

Google Calendar

Use the Calendar to keep track of your meeting and company events.

  • When creating an event, add rooms and people, and use Find a Time to schedule
  • Try out the many plugins by going to Settings > Labs.  Try "Hide morning and night"
  • Use "Suggested times" to quickly find a day where all guests are available

Google Drive

Most of our docs, spreadsheets and presentations live in Google Drive.

  • Make sure you have the Shared Folder added to your My Drive
  • Make sure you save your files someone in there to be able to search

Network Drive

Our network drive is available to share files between departments and people.

  • In Mac Finder, select Go > Connect to Server... and enter smb://shared/storm8 
  • From your browser, go to http://shared.storm8.com for easy read-only access

HipChat

HipChat allows us to communicate quickly with other people and teams.

  • Browse the room directory once in a while and see what's out there
  • Use groups and @ syntax over one-on-one chats to increase visibility
  • Archive old groups if they are no longer used—they just clutter the directory
  • Use emoticons such as "(squirrel)"—https://storm8.hipchat.com/emoticons
  • Use "/code" when pasting code to get proper spacing and code highlighting.

Confluence

Confluence is our new super-powered Wiki!  WYSIWYG editing, FTW!

  • Avoid adding time-sensitive info that can quickly be outdated.  Use links!
  • Check out Atlassian University YouTube videos to learn how to use Jira

JIRA

JIRA is a comprehensive task and bug tracker with support for agile.

  • Go to https://storm8.atlassian.net to log in.
  • Visit YouTube to learn the basics of Jira.

Reviewboard

Reviewboard allows us to do code-reviews remotely and asynchronously.

  • Go to https://wiki.storm8.com/index.php/Review_Board to get started.

Storm8 Test Binaries

Download any of our games in development for fun or play-testing.

  • Go to http://test.storm8.com on your mobile device and browse

iTunes Connect Status

This page displays everything you want to know about our games on the app store.

  • Go to http://itc.storm8.com and try the different tabs and game links

Content Release Calendar

The release calendar is a detailed view into content release scheduling.

  • Go to  https://docs.google.com/a/storm8.com/... and select the Dashboard tab

Storm8 Fun & Culture

Contacts

Need to match a face to a name or find out what department they're in?

  • Go to http://contacts.storm8.com.  Use the search box to dynamically filter text.

Floorplan

Need to know where everyone sits, or just want to look at the building map?

  • Head to http://floorplan.storm8.com to find where everyone sits!
  • Talk to Elizabeth about any moves or changes

Cafe

Want to know what's for lunch or dinner?  Want to vote for your favorite snacks?

  • Head to http://cafe.storm8.com to view our lunch and dinner menus
  • Vote for your favorite lunch and dinner selections, and snacks!

SushiRush

Check out this hackathon project mean to help you match Stormie faces to names!

  • Go to http://peeps.storm8.com/sushirush
